HTML, CSS, and JavaScript

Overview

HTML, CSS, and JavaScript are the core technologies for building websites. HTML structures the content, CSS styles it, and JavaScript adds interactivity. Together, they create dynamic web pages. This page covers their histories and how they function.

History of HTML

HTML (HyperText Markup Language) was created by Tim Berners-Lee in 1989 at CERN. It was designed to share scientific documents over the internet. The first version, HTML 1.0, was released in 1993. It evolved with versions like HTML 2.0 (1995) and HTML 4.01 (1999). HTML5, introduced in 2014, added features for multimedia and better structure. Today, it's the standard for web content.

History of JavaScript

JavaScript was created by Brendan Eich in 1995 at Netscape. Originally called Mocha, then LiveScript, it was renamed JavaScript to ride on Java's popularity. It was standardized as ECMAScript in 1997. JavaScript evolved with versions like ES6 (2015), adding features for modern web apps. It's now used for both client-side and server-side development.

How HTML Works

HTML uses tags to structure web content. For example, <h1> for headings, <p> for paragraphs, and <a> for links. Browsers read HTML and display it as a webpage. It's like the skeleton of a site, defining elements without styling.

  • Basic Structure: Starts with <!DOCTYPE html>, then <html>, <head>, and <body> tags.
  • Elements: Tags like <div> group content, and attributes add details (e.g., href in links).
  • Semantics: HTML5 tags like <header> and <article> make content meaningful for search engines.

How CSS Works

CSS styles HTML elements by applying rules. A rule has a selector (e.g., h1) and declarations (e.g., color: blue). It can be inline, internal, or external. CSS controls layout, colors, fonts, and responsiveness.

  • Selectores: Target elements by tag, class (.class), or ID (#id).
  • Properties: Like margin, padding, background, and flexbox for layouts.
  • Cascading: Rules inherit and override based on specificity and order.

How JavaScript Works

JavaScript is a programming language that runs in the browser. It manipulates HTML and CSS dynamically. For example, it can change text on a button click or fetch data from servers. It's event-driven and uses APIs for interactions.

  • Variables and Functions: Store data and define reusable code.
  • DOM Manipulation: Accesses and changes webpage elements via the Document Object Model.
  • Events: Responds to user actions like clicks or scrolls.
  • Asynchronous Code: Handles tasks like loading data without freezing the page.

How They Work Together

HTML provides structure, CSS adds style, and JavaScript enables interaction. For instance, HTML creates a button, CSS makes it look nice, and JavaScript makes it functional. This trio powers modern web development, from simple blogs to complex apps.
